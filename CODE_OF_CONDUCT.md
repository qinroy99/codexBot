# Code of Conduct

## Our Pledge

我们希望 `codexBot` 社区保持友好、务实、可协作。

## Expected Behavior

- 尊重他人
- 聚焦问题本身
- 提供清晰、可执行的反馈
- 讨论技术时保持克制和礼貌
- 对新手保持耐心

## Unacceptable Behavior

- 人身攻击
- 骚扰、歧视、羞辱
- 恶意泄露隐私或敏感信息
- 发布真实密钥、凭据或用户数据
- 持续性破坏讨论秩序

## Maintainer Responsibility

维护者可以：

- 编辑或删除不合适内容
- 关闭偏离主题或具有攻击性的讨论
- 对持续违规行为采取限制措施

## Scope

本行为准则适用于：

- Issues
- Pull Requests
- Discussions
- 其他与本项目相关的公开协作场景
