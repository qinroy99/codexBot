# Release Notes v0.1.0

发布日期：2026-03-14

## 核心内容

- 发布 `codexBot` 的首个纯 Web 版
- 聚焦 IM bridge 管理与运行监控
- 当前主打 QQ 私聊桥接 + 本地 Web 控制台

## 主要能力

- 渠道配置与测试连接
- bridge 状态控制
- 日志流、doctor、自动修复
- 会话管理、对话查看、消息过滤
- 审计查看与 JSON/CSV 导出
- 告警阈值与浏览器通知

## 已知说明

- 运行需要本机可用 `node`
- 部分功能依赖真实运行目录 `C:\Users\Administrator\.claude-to-im`
- 审计和绑定写操作受本机权限影响
