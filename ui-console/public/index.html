QQ 设置

按你给的参考图，把 QQ 面板拆成“机器人凭据 / 允许用户 / 图片设置 / 设置指南”四块。

机器人凭据

输入你的 QQ 机器人 App ID 和 App Secret。

允许的用户

允许使用桥接的 user_openid，逗号分隔。

留空则允许所有用户。首次联调建议先配置自己的测试账号。

图片设置

控制机器人如何处理来自 QQ 的图片。

设置指南

把最常用的联调步骤直接放在页面里，减少来回切换文档。

  1. 前往 QQ 机器人开放平台 q.qq.com/qbot/openclaw 创建机器人并生成 App ID / App Secret。
  2. 先保存凭据并点击“测试连接”，确认可以获取 access token。
  3. 联调时先填自己的 user_openid，等链路稳定后再放开白名单。
  4. 如果需要收图，开启图片输入并保守设置最大图片大小。
